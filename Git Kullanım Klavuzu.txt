Bir git depolama alaný almak için ;

$ git init

Takip etmek istediðiniz dosyalarý belirtmek için ;

$ git add README

README dosyasýnýn içinde deðiþiklik yapmak için ;

$ git commit -m 'initial project version' (initial project version yazmýþ oluyoruz içine.)

Üzerinde deðiþiklik yapmak için zaten olan "grit" isimli bir dosyayý çekmek için ;

$ git clone git://github.com/schacon/grit.git

Üzerinde deðiþiklik yapmak için zaten olan "grit" isimli bir dosyayý "mygrit" dosyasýnýn içine çekmek için ;
 
$ git clone git://github.com/schacon/grit.git mygrit
